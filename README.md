# JavaScript Functions Project 

This project was completed by two contributors to practice beginner JavaScript functions and parameters. The program includes a `greet` function that displays a personalized greeting message and an `add` function that calculates the sum of two numbers. The results are stored in variables and printed to the console to demonstrate basic function creation, function calls, and returning values in JavaScript.//
